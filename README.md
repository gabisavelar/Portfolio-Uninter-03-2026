# Portfolio-Uninter-03-2026
Este portfólio foi criado com intuito de colocar em prática meus conhecimentos de desenvolvimento de aplicações web, este presente trabalho foi baseado nas metodologias e ensino da Uninter.
Nesta atividade coloquei em prática integrações com Javascript, bem como a visualização de 4 páginas, sendo uma delas o sobre mim com informações iniciais sobre Hobbies, página com informações acadêmicas e extra-curriculares e por fim, uma página para contato.
